How to run:
We used R (version 3.4.1) and RStudio (version 1.0.153).

Run 'package_install.R' to install the packages we use in the scripts.

In 'config.R' set the full paths of the 2 data sets used.

The code for Exercises 1.4, 2.1, 2.2, 2.3, 2.5 is in scripts named accordingly.
Exercise 2.4 has 3 different scripts "script24*.R"

The report is Report.pdf.
